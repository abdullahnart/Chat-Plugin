Instant Chat Assistant is a powerful WordPress plugin that enables **real-time user-to-user chat** on your website. Users can send messages, share files, and even send voice notes without refreshing the page. Ideal for community sites, marketplaces, member portals, and eLearning platforms.

## Installation

1. Download the plugin and upload it to `/wp-content/plugins/`.  
2. Activate the plugin via the WordPress **Plugins** menu.  
3. Add the shortcode `[quick_chat]` to any page or post to display the chat interface.
