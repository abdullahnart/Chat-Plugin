== Description ==

Instant Chat Assistant is a powerful **user-to-user chat plugin** for WordPress that enables your site members to communicate instantly.  
It includes **separate upload buttons** for images and documents, supports **audio messages**, and delivers **real-time notifications** for new messages.

=== 🌟 Key Features ===

💬 Live Chat & Real-time Messaging – Chat instantly with registered users with real-time updates.
🖼️ Image Uploads – Share images instantly in the chat.
📄 Document Uploads – Send PDF, DOC, and other files with a dedicated upload button.
🎤 Audio Uploads – Record and send voice notes.
🔔 Instant Notifications – Get notified immediately when a new message arrives.
📱 Responsive Design – Works perfectly on desktop, tablet, and mobile.


=== Use Cases ===

🛎️ **Customer Support Chat** – Provide instant help to your customers directly from your website.  
🔄 **Peer-to-Peer Marketplace** – Allow buyers and sellers to communicate, share product images, and send invoices or documents.  
📚 **Online Classes & E-Learning** – Let teachers share lecture notes, assignments, and voice explanations with students.  
🤝 **Team Collaboration** – Enable teams to exchange ideas, share files, and discuss projects in real-time.  
📅 **Event Coordination** – Facilitate communication between event organizers and participants before, during, and after events.  


== Installation ==
1. Download the plugin from WordPress.org.
2. Upload the plugin folder to `/wp-content/plugins/` using an FTP client or the WordPress Plugin Installer.
3. Activate the plugin via the 'Plugins' menu in WordPress.
4. Use the shortcode `[quick_chat]` to display the chatbot on any page or post.

== Frequently Asked Questions ==

= Do users need to be logged in to chat? =
Yes, only registered and logged-in users can use the chat feature.

= Does the chat work in real-time? =
Yes, messages, file uploads, and notifications are updated instantly without refreshing the page.

= Can I upload image and document both? =
Yes, you can upload images and documents. The chat interface includes two dedicated buttons — one for uploading images and another for uploading documents — making file sharing more organized and user-friendly.

= Is this plugin mobile-friendly? =
Yes, Instant Chat Assistant is fully responsive and works on all devices.

= Does this plugin include a separate login or signup form? =
No. The "Login or Register" functionality uses the default WordPress login/registration system. There is no separate form included — users will be redirected to WordPress's default login/registration page.
